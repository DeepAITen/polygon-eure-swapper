# Polygon EURe Swapper

Système de conversion automatique entre stablecoins (USDT, USDC) et EURe sur Polygon.

## Features
- Swap USDT/USDC → EURe
- Swap EURe → USDT/USDC
- Vault sécurisé pour private key
- Interface simple pour transactions

## Stack
- Node.js + ethers.js
- Polygon network
- Uniswap/QuickSwap DEX
